# Testirovanie-urtk1
ТЗ: Создать простейший калькулятор в консольном приложении на языке c#.

Калькулятор должен принимать с клавиатуры значения двух переменных и выполнять с ними одну из четырёх операций:
1. Сложение (+)
2. Вычитание (-)
3. Деление (/)
4. Умножение (*)

В калькуляторе должны быть следующие проверки:

1. Проверка на деление на 0:
   
   Должно выводиться сообщение о невозможности деления.

2. Проверка на выбор несуществующей операции:

   Должно выводиться сообщение о выборе несуществующей операции.


TEST CASE:

1) Ввод первой переменной "5"
   Ввод второй переменной "6"
   Выбор операции (+)
   Вывод: 11

   Ввод второй переменной "3"
   Выбор операции (+)
   Вывод: 12

   Выбор операции (-)

4) Ввод первой переменной "15"

   Выбор операции (-)
   Вывод: 7

   Выбор операции (/)
   Вывод: 3

6) Ввод первой переменной "20"
   Ввод второй переменной "5"
   Выбор операции (/)
